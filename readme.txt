Asas Al Waha Technical Services LLC - Dark Premium Website
=========================================================

This package contains a PREMIUM dark-theme static website.

Files & Structure
-----------------
- index.html       -> Home page (hero with premium card, ribbon section, services, CTA)
- about.html       -> About page
- services.html    -> Services page (full list)
- projects.html    -> Sample projects / references
- contact.html     -> Contact & map

- /assets/styles.css    -> Main styling (dark, teal + gold, gradients, shadows, animations)
- /assets/scripts.js    -> Mobile nav + scroll reveal animations
- /assets/logo.svg      -> Logo used in header and favicon

How to Deploy on Netlify
------------------------
1. Zip this entire folder (asasalwaha_dark_premium) into a single ZIP file.
2. On Netlify, choose "Add new site" -> "Deploy manually".
3. Upload the ZIP file.
4. Netlify will provide a live URL.
